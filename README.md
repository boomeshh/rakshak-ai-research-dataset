# Rakshak AI - Seeded Threat Dataset

This repository contains simulated and seeded cyber threat datasets used for testing and evaluation of the Rakshak AI Defence Cyber Safety Portal.

## Included Samples
- Phishing URLs
- Scam SMS messages
- Fake defence impersonation messages
- APK threat samples
- Synthetic complaint records
- Dashboard analytics testing data

## Purpose
Used for:
- Threat classification testing
- Risk scoring validation
- CERT escalation workflow testing
- AI-assisted phishing analysis
- Dashboard and audit log evaluation

## Disclaimer
All datasets are simulated and created strictly for educational and research purposes.
No real defence or personal data is included.
